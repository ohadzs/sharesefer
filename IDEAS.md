# ShareSefer — ideas / backlog

Future features, not yet built. Roughly ordered by appeal.

## Discovery
- **Near me / קרוב אליי** — show books available close to you (by city → neighborhood → distance). Needs a location field on profiles (city at least). Big value for a lending site: people lend to neighbors.
- **Map view** — pin available books on a map.
- **Filter / sort** — by language (Hebrew/English), genre, year, distance, newest.

## Personal
- **Favorites / מועדפים** — heart a book to save it; a "my favorites" list.
- **Looking for / מחפש** — a wishlist: post a book you want. Owners see demand; you get notified when someone lists it nearby.
- **Recommendations for me / המלצות בשבילי** — suggest books from what you've read / favorited / borrowed. (Could seed from Ohad's `books.md` scores.)
- **My shelf** — a shareable public link to everything you lend (`/u/<name>`), good for SEO too.

## Trust & flow
- **Listing status** — available / lent-out, with an easy "mark returned".
- **Lender reputation** — light rating or "borrowed N times" badge so people trust strangers.
- **Notifications** — email/WhatsApp when a wanted book appears or someone requests yours.

## Stretch
- **Reading lists / themed shelves** — curated collections (e.g. "פילוסופיה למתחילים").
- **QR for a physical "little library"** — print a code on a home shelf that opens its listings.

> Add freely — capture first, prioritize later.
